# skills/

Each subdirectory is one skill, consumable by any Agent-Skills-compatible runtime.

```
skills/
  <skill-name>/
    SKILL.md          # YAML frontmatter (name, description, …) + the body
    resources/        # optional, loaded on demand
    scripts/          # optional helper scripts
```

See the [repository README](../README.md#skill-format) for the format and the
[contributing guide](../CONTRIBUTING.md) for the quality bar. New skills land here via
pull request with maintainer review.
